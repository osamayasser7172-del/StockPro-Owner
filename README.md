# StockPro — ملف المالك 👑

## التشغيل
افتح `admin.html` في المتصفح — مفيش سيرفر مطلوب! ✅

## بيانات الدخول
| البيان | القيمة |
|--------|--------|
| مفتاح المالك | `STOCKPRO-OWNER-2024` |
| يوزر الأدمن | `admin` |
| باسورد الأدمن | `admin123` |
| سريال المالك | `SP-MSTR-OWNR-2024-SPRO` |

## الملفات
- `admin.html` — لوحة التحكم (إدارة العملاء والتراخيص)
- `stockpro.html` — تطبيق العميل (للتجربة على جهازك)

## طريقة العمل
1. افتح `admin.html` → سجل دخول → أضف عميل جديد
2. انسخ مفتاح الترخيص اللي اتولد
3. ابعته للعميل مع ملفات مجلد `StockPro-Client`
4. العميل يفتح `stockpro.html` ويدخل المفتاح

## ⚠️ مهم
- الأدمن والعميل **لازم يكونوا على نفس الجهاز** علشان البيانات مشتركة في localStorage
- لو عايز كل عميل يكون على جهاز لوحده، ابعتله المفتاح ومعاه stockpro.html
